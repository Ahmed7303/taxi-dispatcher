# Такси Диспетчер

Десктоп-пульт диспетчера службы такси (Ашхабад). Electron + React + Vite.
Подключается к бэкенду по адресу `https://taxi.achar7303ai.com`.

## Возможности (текущий этап)
- Вход диспетчера по логину и паролю (JWT, авто-обновление токена)
- Экран «Заказы»: снимок текущих заказов + живая лента событий в реальном времени (Socket.io)
- Тёмная тема, интерфейс на русском

В разработке: водители и пополнение токенов, модерация ВИП-заявок, споры, клиенты.

> По бизнес-правилу диспетчер **не назначает и не перенаправляет** заказы —
> только наблюдает и обслуживает токены/модерацию (анти-коррупционное ограничение).

## Требования
- Node.js 18+ (проверено на 22)
- Windows / macOS / Linux

## Запуск в режиме разработки
```bash
npm install
npm run dev
```
Поднимется Vite (порт 5173) и окно Electron подключится к нему.

## Сборка установщика под Windows
```bash
npm run dist
```
Готовый установщик появится в папке `release/`.

## Структура
```
electron/        главный процесс Electron + preload
src/
  api/           клиент бэкенда (JWT/refresh) и Socket.io
  components/    оболочка пульта (сайдбар, статус)
  screens/       экраны: вход, заказы
  store/         состояние (zustand)
  styles/        тёмная тема
```

## Конфигурация
Адрес бэкенда задаётся в одном месте — `src/api/client.js`, константа `BASE_URL`.
